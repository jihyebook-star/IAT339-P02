Style Guide

Interactive Elements

In-text link

CSS selector:

a

Sample code:

<p>This is a paragraph with an <a class = "inTextLink" href="#">in-text link</a> inside it.</p>

Rendered element:

This is a paragraph with an in-text link inside it.

Navigation Link

CSS selector:

.checkout-btn

Sample code:

<a href="#" class="checkout-btn">CHECKOUT</a>

Rendered element:

CHECKOUT

Submit/Reset Buttons

CSS selector:

.submit-btn, .reset-btn

Sample code:

<a href="#" class="submit-btn">Submit Button</a> <a href="#" class="reset-btn">Reset Button</a>

Rendered element:

SUBMIT RESET

Text Input Field with Label

CSS SELECTOR:

contact-form, label, input

Sample code:

<label for="name">FULL NAME</label> <input id="name" type="text">

Rendered element:

SUBMIT

Radio Button Input Field with Label

CSS selector:

input[type="radio"], label

Sample code:

<input type="radio" id="pickup" name="delivery"> <label for="pickup">Pick up in store</label>

Rendered element:

Image as a Link

CSS selector:

.products-grid, .product-card, .product-overlay, .view-btn, .product-info

Sample code:

<div class="products-grid"> <div class="product-card"> <img src="img/SweetStrawberryCheese.png" alt="Sweet Strawberry Cheese"> <div class="product-overlay"> <a href="strawberry-product-listing.html" class="view-btn">VIEW DETAIL</a> <div class="product-info"> <h3>Sweet Strawberry Cheese</h3> <p>$42.99</p> </div> </div> </div> </div>

Rendered element:

VIEW DETAIL

Sweet Strawberry Cheese

$42.99

h1, h2, h3, h4, h5, p

Sample code:

<h1>Heading Level 1</h1> <h2>Heading Level 2</h2> <h3>Heading Level 3</h3> <h4>Heading Level 4</h4> <h5>Heading Level 5</h5> <p>Paragraph</p>

Rendered element:

Heading Level 1

Heading Level 2

Heading Level 3

Heading Level 4

Heading Level 5

Paragraphs

Paragraphs

CSS selector:

p

Sample code:

<p> A basic template for building my own style guides, to be utilized in the future. This is a paragraph for the paragraph element section! </p>

Rendered element:

A basic template for building my own style guides, to be utilized in the future. This is a paragraph for the paragraph element section!

Numbered Lists

CSS selector:

ol, li

Sample code:

<ol> <li>Choose a cake type</li> <li>Select / Customize your design</li> <li>Review & checkout</li> </ol>

Rendered element:

  1. Choose a cake type
  2. Select / Customize your design
  3. Review & checkout

Interactive Elements

In-text link

CSS selector:

a

Sample code:

<p>This is a paragraph with an <a class = "inTextLink" href="#">in-text link</a> inside it.</p>

Rendered element:

This is a paragraph with an in-text link inside it.

Navigation Link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Default Button</a>

Rendered element:

Default Button

Submit/Reset Buttons

CSS selector:

.button-submit, .button-reset

Sample code:

<a href="#" class="button-submit">Submit Button</a> <a href="#" class="button-reset">Reset Button</a>

Rendered element:

Submit Button Reset Button

Text Input Field with Label

CSS selector:

input[type="text"], label

Sample code:

<label for="name">Name:</label> <input type="text" id="name" name="name">

Rendered element:

Radio Button Input Field with Label

CSS selector:

input[type="radio"], label

Sample code:

<input type="radio" id="pickup" name="delivery"> <label for="pickup">Pick up in store</label>

Rendered element:

Brand Description

Cake, Beyong Your Imagination is a handmade cake brand that makes custom cakes for special occasions. Customers can commission cakes with personalized designs tailored to their needs, either by describing their ideas with the help of an AI-assisted design preview or by selecting from the existing cake designs. As a brand, the company is presented as thoughtful, creative, and detail-oriented, valuing each personal connection and care in every order. The studio approaches each cake as a special part of the celebration rather than a mass-produced product. Customers are choose Jihye's Cake Crumb Studio because of its flexibility, personalization, and a more intimate ordering experience. The ability to customize designs ensures that each cake feels unique and meaningful for the occasion, making the studio a reliable choice.
